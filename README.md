Đây là một ví dụ về bài tập kiểm thử đơn vị với Java.
### Tính năng:
- Cộng hai số nguyên.
- Trừ hai số nguyên.
- Nhân hai số nguyên.
- Chia hai số nguyên với xử lý lỗi chia cho 0.

### Yêu cầu trước khi sử dụng:
- Java Development Kit (JDK) 8 hoặc mới hơn.
- Công cụ build Maven.
- JUnit 5 để kiểm thử.
